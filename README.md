# MySocialApp
> 인스타 클론을 바탕으로 나만의 소셜 앱 만들기  

</br>

##1. 제작 기간 및 참여 인원
- 2021.09 ~ 2021.10(약 2달, 일시중단)
- 개인 프로젝트

</br>

##2. 사용 기술
- Swift
- SwiftUI
- Firebase

</br>

##3. 핵심 기능
- 소셜 로그인
- 게시물 작성 및 개인 컨텐츠 공유
- 커뮤니티
